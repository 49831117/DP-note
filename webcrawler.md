## 入門
### HTML
HyperText Markup Language
- 標籤（tags），如：`<p>`、`</p>`
- 屬性（Attributes），如：`<img>` 的 `src`、`width`、`height`
  `<img scr="sample.jpg" width="20" height="30">`
- HTML5 支援跨平台，擴充及改進 HTML 標籤和 API。
    ```html
    <!DOCTYPE html>
    <html lang="zh">
    <head>  
    <meta charset="utf-8">
    <title>網頁標題文字</title>
    </head>
    <body>
    網頁內容
    </body>
    </html>
    ```

- XML 非取代 HTML，描述資料用。補足 HTML 標籤的不足，更有擴充性
